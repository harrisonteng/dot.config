NAME

rain -- Let it rain.

SYNOPSIS

rain

DESCRIPTION

rain simulates rain.

OPTIONS

rain does not accept options.

ENVIRONMENT

The working of rain is not influenced by any environment variables.

BUGS

rain does not have any known bugs.
