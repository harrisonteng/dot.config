NAME

true - Exit succesfully. false - Exit unsuccesfully.

SYNOPSIS

(true | false) [OPTION]

DESCRIPTION

true exits succesfully. false exits unsuccesfully.

OPTIONS

true and false accept the following options:

--help

Print out a short help message, then exit.

--version

Print out its version number, then exit.

ENVIRONMENT

The working of true and false are not influenced by any environment variables.

BUGS

true and false have no known bugs.
