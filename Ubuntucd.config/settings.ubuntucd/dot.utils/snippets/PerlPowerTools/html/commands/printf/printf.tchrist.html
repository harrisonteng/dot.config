NAME

printf - format and print data

SYNOPSIS

printf format [ argument ... ]

DESCRIPTION

The printf command uses the first argument as the format that describes how to print the remaining arguments. Unlike the standard printf(1) command, this one uses the Perl version, which means that the \v escape is not supported, but various other things are. See "sprintf" in perlfunc for details.

RESTRICTIONS

This command should not be used in setuid programs as it does not run untaint its argments and will trigger errors like Insecure dependency in eval while running setuid at /opt/ppt/bin/printf line 16.

SEE ALSO

printf(3), "sprintf" in perlfunc
