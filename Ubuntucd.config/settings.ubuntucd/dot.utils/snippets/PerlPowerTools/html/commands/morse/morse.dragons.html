NAME

morse - Translate standard input into morse code

SYNOPSIS

morse [-s]

DESCRIPTION

morse takes standard input and translates it into morse code. Unkown characters are ignored.

OPTIONS

morse only takes one option:

-s

Print the morse code as '.'s and '-'s, rather than 'dit's and 'daw's.

ENVIRONMENT

morse is not affected by any environment variables.

COMPATIBILITY

This version offers a slightly larger set of characters than the BSD version.

BUGS

morse has no known bugs.
