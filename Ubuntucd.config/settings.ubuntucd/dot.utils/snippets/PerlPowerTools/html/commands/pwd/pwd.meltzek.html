NAME

pwd - working directory name

SYNOPSIS

pwd

DESCRIPTION

Pwd prints the pathname of the working (current) directory.

OPTIONS

pwd takes no options.

ENVIRONMENT

The working of pwd is not influenced by any environment variables.

BUGS

pwd has no known bugs.
